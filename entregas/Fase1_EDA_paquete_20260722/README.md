## Cómo usar el notebook

1. Kernel Python 3.10+ con: `pymupdf`, `pydantic`, `pandas`, `matplotlib`, `seaborn`.
2. Ajustar `PROJECT_ROOT` en la primera celda de código si la ruta del repo es distinta.
3. Los CSV de este paquete ya están listos en `data/`; el notebook también puede regenerarlos si existen PDF en `data/raw/pdfs` del repo completo.
